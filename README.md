# Family Money Tracker

Tento projekt slúži na sledovanie rodinného majetku, príjmov a výdavkov.

## Spustenie projektu

Najprv spustite vývojový server:

```bash
npm run dev
```

Otvorte [http://localhost:3000](http://localhost:3000) vo vašom prehliadači.

## Technológie
- **Next.js**
- **Supabase**
- **Tailwind CSS**
- **Framer Motion**
